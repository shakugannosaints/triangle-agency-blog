# 三角机构博客设计研究

## AgencyOS 设计风格分析

### 核心特征
1. **冷峻企业风格**：深色背景（深蓝/深灰），霓虹色强调（青色、洋红、琥珀色）
2. **多主题支持**：
   - **Default (Night)**：深蓝背景 + 青色/洋红/琥珀色强调
   - **Day**：浅色背景 + 红色强调
   - **Win98/Retro**：复古操作系统风格
   - **Fluent**：现代Windows风格

### 颜色系统
- **深色主题**：
  - 背景：#0D0D0F (深灰)
  - 面板：#101120 (深蓝)
  - 边框：#262A3B
  - 强调色：
    - 青色：#07F0FF (霓虹青)
    - 洋红：#FF0F9A (霓虹粉)
    - 琥珀：#F5B54C (温暖金)
    - 柔和：#8C93B0 (灰蓝)

- **浅色主题**：
  - 背景：#F8F6F4 (米白)
  - 面板：#FFFFFF
  - 边框：#E0D6D6
  - 强调色：
    - 青色 → 红色：#D7263D
    - 洋红 → 深红：#8C1C30

### 排版
- **字体**：Inter (现代无衬线) + IBM Plex Mono (等宽)
- **风格**：现代、极简、高对比

### 交互元素
- **阴影**：`0 0 0 1px rgba(7, 240, 255, 0.16), 0 10px 30px rgba(0, 0, 0, 0.26)`
- **动画**：
  - `emergency`: 色相旋转 + 饱和度脉冲 (0.8s)
  - `pulseGrid`: 不透明度脉冲 (3s)
  - `slideInFromRight`: 从右滑入 (0.3s)
  - `fadeIn/fadeOut`: 淡入淡出 (0.2s)

### 设计哲学
- **功能优先**：信息密度高，布局紧凑
- **视觉反馈**：强调交互状态，使用颜色和动画
- **企业感**：专业、可靠、高科技

## 三角机构 视觉元素分析

### 标志性元素
1. **红色三角形**：
   - 颜色：#D7263D (鲜艳红色) 或 #E63946 (深红)
   - 几何：等边三角形，作为品牌核心
   - 用途：Logo、强调、装饰

2. **配色方案**：
   - **主色**：深蓝 (#1A1A3E) + 鲜红 (#D7263D)
   - **辅助色**：
     - 青色 (#07F0FF) - 科技感
     - 黄色 (#FFD700) - 警告/异常体
     - 白色/浅灰 - 背景对比

3. **视觉氛围**：
   - **暗黑科幻**：深色背景，霓虹色强调
   - **诡异神秘**：异常体的超现实视觉
   - **机构感**：几何图形、网格、线条
   - **危险感**：红色三角形的警告意味

### 设计元素
- **几何图形**：三角形、网格、线条
- **质感**：光泽感、发光效果、扭曲变形
- **动感**：流动、脉冲、变换

## 博客设计方向

### 融合策略
1. **继承AgencyOS的企业冷峻风**：深色主题、霓虹色强调、现代排版
2. **融入三角机构的标志元素**：红色三角形、深蓝配色、科幻氛围
3. **强化沉浸感**：
   - 视差滚动效果
   - 过渡动画
   - 几何装饰元素
   - 红色三角形的战略性使用

### 核心配色
- **背景**：#0D0D0F (深灰) 或 #1A1A3E (深蓝)
- **文本**：#E8E8E8 (浅灰) / #FFFFFF (白色)
- **强调色**：
  - 主强调：#D7263D (三角机构红)
  - 次强调：#07F0FF (霓虹青)
  - 警告：#FFD700 (黄色)

### 排版系统
- **标题**：大胆、有存在感，使用深蓝色或红色
- **正文**：高可读性，使用浅灰色
- **强调**：红色三角形或霓虹色

### 交互设计
- **悬停效果**：红色三角形闪烁、颜色变化
- **过渡动画**：文章卡片滑入、淡入淡出
- **视差滚动**：背景与前景分离，增强深度感
- **几何装饰**：三角形网格、线条动画

---

## 实现优先级

1. **第一阶段**：基础布局 + 深色主题 + 红色三角形强调
2. **第二阶段**：文章列表 + 分类/标签系统
3. **第三阶段**：沉浸式交互 (视差、动画、过渡)
4. **第四阶段**：GitHub Pages 部署配置
